## 💪 Health-to-Earn (H2E)

FitRudrah's Futurecoin (FTC) is a Solana-powered Health-to-Earn ecosystem that rewards users for verified physical activity.

### Core Concept

🔥 **1 Calorie Burned = 1 Fitcoin (FTC)**

Transform your daily fitness into digital rewards.
